## Scripts

```sh
# build and run
$ yarn start
$ yarn start:prod

# run from dist
$ yarn start:dist

# building
$ yarn build
$ yarn build:prod
$ yarn build:watch

# testing
$ yarn test
$ yarn test:watch
$ yarn lint

# commitizen
$ yarn commit
```
